Owner / Representative Director

増田 早紀

株式会社SOTTO 代表取締役

一級建築士 宅地建物取引士 大家 女性起業家 元行政・建築職

元行政の建築職として公共施設の設計・工事監理や建築許認可、不動産相談に携わった後、独立。建築と不動産の視点から空き家活用をプロデュースし、建築デザインや不動産伴走支援を行っています。

起業という選択は自由である一方で、孤独や不安と隣り合わせでもあります。私自身も起業初期には手探りで人とのつながりを築き、自宅住所での登記に不安を感じバーチャルオフィスを利用した経験があります。そうした経験から、同じように挑戦する方々が安心して活動できる環境の必要性を強く感じるようになりました。

すずらんテラスは、地域に眠る空き家を活用し、鈴蘭台で活動する女性起業家・フリーランスの方々が安心して働ける拠点をつくりたいという想いから生まれました。常設シェアオフィスに加えバーチャルオフィス機能も備え、月1〜2回は地域の作家や起業家によるイベント・勉強会も開催予定。挑戦する人同士が支え合い、新しい可能性が生まれる場所を目指しています。
